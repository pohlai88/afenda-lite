## When reopening (checklist from `fumadocs-i18n`)

Use the vendor skill as a **method library** after an Approved slice — adapt to Lite, do not paste Xerp 7-locale packs.

```text
[ ] lib/i18n.ts — locales + default
[ ] middleware — createI18nMiddleware (docs app only; not apps/web proxy.ts)
[ ] app/[lang]/docs/[[...slug]] — move routes
[ ] source.ts — pass i18n into loader
[ ] content/docs/<locale>/ — split trees; sidebar per locale
[ ] OpenAPI generator — decide EN-only ops vs localized titles (Lite default: EN-only ops)
[ ] No product secrets on docs project (Day-1 rule still holds)
```

**Default Lite choice when reopened:** keep OpenAPI operation pages English-only; localize narrative MDX first.
